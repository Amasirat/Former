# Requirements Specifications

## Create Form Definitions
* Creating Form fields
    * Fields of any type
    * Any type of validation rules
    * Regex
    * Conditional Behaviors, both intrafields and interfields
* Saving created form fields for use in other forms
* Editing saved fields
* Specify permissions for who can fill this form

## Show Forms
* Store Defined forms and give back links to view them
* Show Form to Authorized or Unauthorized users

## Handle Submissions
* Take and Store submissions for any defined form
* Validate submissions
* Retrieve submissions -- done by authorized users

## Authentication 
* 2-Step Verification
* OAuth Support
* Reset Password
* Token Service
